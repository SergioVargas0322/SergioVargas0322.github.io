# CursosCISCO

Catalogo web para organizar cursos de Cisco por modulos y temas.

## Estado inicial
- Proyecto nuevo.
- Solo incluye el primer curso:
  - `Conceptos Basicos de Hardware de Computadora`
- Base preparada para escalar a ~14 cursos.

## Estructura
- `index.html`: catalogo principal.
- `assets/css/styles.css`: estilos globales del catalogo.
- `assets/js/data.js`: datos de cursos (por ahora 1 curso).
- `assets/js/app.js`: render, busqueda y detalle de modulos/temas.
- `assets/images/hardware-basics/`: imagenes del primer curso.

## Uso local
Abrir `index.html` en el navegador.
